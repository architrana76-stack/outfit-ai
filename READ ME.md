# 👕 Outfit AI

A smart outfit recommendation web app that suggests clothing based on real-time weather conditions.

## 🌐 Live Demo
👉 https://architrana76-stack.github.io/outfit-ai/

## 🚀 Features
- 🌦 Weather-based outfit suggestions
- 👕 Separate top & bottom clothing display
- 🎨 Smart color matching
- ❤️ Save outfits locally
- 📱 Responsive UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API

## ⚙️ Setup
1. Clone repo
2. Open `index.html`
3. Add your API key in `script.js`
