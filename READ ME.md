# 👕 Outfit AI

A smart outfit recommendation website that suggests clothes based on real-time weather conditions.

## 🚀 Features
- 🌦 Weather-based outfit suggestions
- 👕 Separate top & bottom clothing display
- 🎨 Smart color matching
- ❤️ Save outfits locally
- 📱 Responsive design

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API

## ⚙️ How to Run
1. Download or clone this repo
2. Open `index.html` in browser
3. Allow location access
4. Click **Generate Outfit**
